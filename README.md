# 元旦快乐网页

这是一个庆祝2026年元旦的网页，包含倒计时、愿望墙和音乐播放功能。

## 音乐播放功能

### 添加本地音乐

1. 准备一个名为`music.mp3`的音频文件
2. 将该文件放在项目根目录下（与index.html、style.css和script.js同一目录）
3. 重新加载网页，音乐播放器将自动检测并播放该文件

### 音乐播放器控制

- 点击▶️/⏸️按钮可以播放/暂停音乐
- 点击🔊/🔇按钮可以静音/取消静音
- 点击进度条可以跳转到指定位置

### 注意事项

- 请确保音频文件格式为MP3
- 建议音频文件大小适中（小于5MB）以保证加载速度
- 某些浏览器可能需要用户交互后才能自动播放音频，此时请点击播放按钮手动开始播放

## 其他功能

- 倒计时显示：显示2026年元旦已经到来的天数
- 愿望墙：可以写下并分享新年愿望
- 节日装饰：雪花、气球和彩带动画效果
